# 24-Hour Timer Selector

A user-friendly web application for selecting and sharing time ranges on a 24-hour circular clock. Designed with elderly users in mind, featuring large text, high contrast colors, and intuitive interactions.

## Features

- **Interactive 24-Hour Clock**: Beautiful circular clock interface with clear hour markers
- **Easy Time Selection**: Click and drag to select time ranges
- **Multiple Ranges**: Select multiple time periods on the same clock
- **Share as Image**: Generate and download a PNG image to share with others
- **Elderly-Friendly Design**:
  - Large, readable text
  - High contrast colors
  - Big, easy-to-click buttons
  - Clear visual feedback
  - Touch-friendly for tablets

## How to Use

1. **Open the Application**: Simply open `index.html` in any modern web browser
2. **Select Time Ranges**:
   - Click on the clock where you want your time range to start
   - Hold and drag to where you want it to end
   - Release to create the time range
3. **Manage Ranges**:
   - View all selected ranges in the panel on the right
   - Click "Remove" next to any range to delete it
   - Click "Clear All" to start over
4. **Share**:
   - Click "Save Image" to download a PNG file
   - Share the image via email, messaging apps, or print it

## Technical Details
